# markit
markit is a a todo list app with advanced features, such as pin to top todo background colors delete and edit todos. Built with core data.

**Features: **

- Add todos 
- Add todos background color
- edit/ deleye todos 
- pin todos to top

**Tech Used:**

- SWIFTUI / SWIFT
- CORE DATA
- MVVM
